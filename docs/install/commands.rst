1.5 NEMAT Commands
==================

Usage: ``nemat <command> [options]``

+----------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+-------------------------------+
| Command        | Description                                                                                                                                                                                      | Use example                   |
+================+==================================================================================================================================================================================================+===============================+
| start          | Prepare current **empty** directory for a NEMAT run. Creates default input.yaml and empty folders for ligands and proteins along with default mdppath and membrane.                              | :ref:`setup <setup>`          |
+----------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+-------------------------------+
| prep           | Assembles the system.                                                                                                                                                                            | :ref:`Preparation <run_nemat>`|
+----------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+-------------------------------+
| prep_<step>    | Prepare input files for a specific step (min, eq, md, ti).                                                                                                                                       | :ref:`Preparation <run_nemat>`|
+----------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+-------------------------------+
| check_<step>   | Checks logs/step.err for GROMACS errors for step (prep, min, eq, md, ti, analyze).                                                                                                               | :ref:`Checking <run_nemat>`   |
+----------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+-------------------------------+
| run_<step>     | Submits job array to run the corresponding step (min, eq, md, ti).                                                                                                                               | :ref:`Execution <run_nemat>`  |
+----------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+-------------------------------+
| s_<step>       | Verifies whether the GROMACS run was successful for step (min, eq, md, ti).                                                                                                                      | :ref:`Checking <run_nemat>`   |
+----------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+-------------------------------+
| analyze        | Analyzes results and produces plots.                                                                                                                                                             | :ref:`Analysis <analysis>`    |
+----------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+-------------------------------+
| img            | Generates result images from existing results_summary.csv files.                                                                                                                                 | :ref:`Analysis <analysis>`    |
+----------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+-------------------------------+
| val            | Displays validation overlap checks from the analysis log.                                                                                                                                        | :ref:`Analysis <analysis>`    |
+----------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+-------------------------------+
| new            | Creates a new run by removing previous input and working files (confirmation prompted).                                                                                                          | :ref:`Other features <other>` |
+----------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+-------------------------------+
| copy           | Copies the current workpath into a new directory. Useful when reusing dynamics with different transitions. Prompts for destination and copy level (eq, md, all).                                 | :ref:`Other features <other>` |
+----------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+-------------------------------+
| update         | Updates NEMAT parameters in current directory to match input.yaml.                                                                                                                               | :ref:`Other features <other>` |
+----------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+-------------------------------+
| clean          | Cleans backup files inside the workPath (confirmation prompted).                                                                                                                                 | :ref:`Other features <other>` |
+----------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+-------------------------------+
| help           | Displays the help message.                                                                                                                                                                       |                               |
+----------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+-------------------------------+
